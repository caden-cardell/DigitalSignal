DigitalSignal Sandbox

Example use: 
>>> d = DS([1])              # Kronecker delta function
>>> x = DS([-1, [0], 1, 2])  # {-1, ➔0, 1, 2}
>>> x(-1)                    # print x[n-1]
DigitalSignal([-1] 0 1 2)
>>> x @ d(-1)                # print x[n]∗δ[n-1]
DigitalSignal([-1] 0 1 2)
>>> x(-1) == x @ d(-1)       # x[n-1] is x[n]∗δ[n-1]
True
